# Markdown 文档模板与示例

## 文首完整模板（复制即用）

```markdown
# 《文档标题》

**成文日期**：YYYY-MM-DD
**最后修订**：YYYY-MM-DD

（此处可写 1～2 句文档目的与存放路径。）

**阅读时当前系统实现可能已发生变化，请以实际代码与产品行为为准，谨慎参考。**

---

## 一、第一节
```

## 实际示例（摘自本项目）

- `docs/prd/Langgraph-superchat-new/Studio集群语义与agent_id设计补充说明.md`：文首含成文日期 + 免责说明。
- `docs/prd/Langgraph-superchat-new/Studio测试请求候选Agent隔离与建图路径说明.md`：成文日期 +「阅读时系统实现可能已经与本文档不一致，谨慎参考。」
